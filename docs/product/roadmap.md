# Roadmap

1. **Foundation:** offline Markdown import, evidence-backed manual findings, trace display, portable report exports, deterministic tests.
2. **Interactive workbench:** layered investigation timeline, contextual details, slash commands, focus/theme roles, active-view saves, data grids and charts, a read-only code browser, rich Markdown with Mermaid/D2 views, then saved views, append-only processing records, workspace migrations, and content-type expansion.
3. **Replaceable analysis:** subprocess engine protocol and a Graphify adapter behind contract tests; operation remains complete without it.
4. **Explicit connectors:** opt-in Git, documentation, issue, and CI adapters with redaction, sync previews, and network policy.
5. **Optional intelligence:** user-configured local or external LLM/embedding providers, consent gates, provenance, budgets, and deterministic non-LLM alternatives.
6. **Distribution:** signed source archives and binaries, checksums, SBOMs, and documented format stability. Release automation is deliberately deferred until versioning and signing policy are accepted.

Tree-sitter, dataframe/chart libraries, and dynamic plugin frameworks require demonstrated use cases and ADRs; none belongs in the foundation.
