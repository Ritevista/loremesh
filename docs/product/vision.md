# Product vision

LoreMesh helps engineers investigate systems using local evidence they can inspect, trace, correct, and reuse. Its differentiator is trustworthy derivation: a finding is useful only when a person can follow it back to immutable source material and understand the processing that produced it. Optional AI may assist later, but it never owns report structure.

The product starts as an offline modular monolith with a dashboard TUI. It should grow through explicit adapters, not by contaminating the domain with vendor concepts. Success means useful manual investigation without optional engines, deterministic portable workspaces, transparent evidence, and exports that remain useful outside LoreMesh.

Near-term users are engineers investigating code, documentation, logs, and CI output on their own machine. The project does not promise collaborative hosting, autonomous conclusions, or comprehensive connector coverage in its foundation phase.
